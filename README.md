# employee-leave-pattern-analyzer

## Project description
Created a system to analyze leave usage by employees and identify frequent leave patterns.


## Key focus areas

* Used Map to store and access the data
* Simple date-based logic using `LocalDate`
* Data aggregation and processing

## Methods Implemented

- Add a new employee
- Add leave details for an employee
- View total leave count per employee
- Identify employees frequently taking long weekend leaves
- View monthly leave summary
- View employees sorted by total leave count
